# NxtTrendzApp

An eCommerce web application inspired by platforms like Amazon and Flipkart.  
This project allows users to browse products, search items, filter products, view product details, and manage cart functionality with authentication.

# 📌 Features

- 🔐 User Authentication (Login & Logout)
- 🛒 Add to Cart Functionality
- ➕ Increment & ➖ Decrement Cart Items
- ❌ Remove Individual Cart Item
- 🗑️ Remove All Cart Items
- 🔍 Product Search
- 🎯 Product Filters
- 📱 Fully Responsive Design
- 🚫 Protected Routes
- 📦 Product Details Page
- 🧾 Cart Summary Section

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/virendrasahu/NxtTrendzApp.git
```

## 2️⃣ Navigate to Project Folder

```bash
cd NxtTrendzApp
```

## 3️⃣ Install Dependencies

```bash
npm install
```

## 4️⃣ Start Development Server

```bash
npm start
```
